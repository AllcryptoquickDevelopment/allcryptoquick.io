Particl Project
Status Report

For general overview of Particl Project's goals visit Project Roadmap

Updated 13 August 2019

Recently completed

  • Particl Marketplace
    • Open Marketplace (beta) released on mainnet
    • OMP Library integration
      • Anon escrow integration
    • Updating/enhancements to automated tests running in the background
    • Added longer retention on free SMSG
    • Bug fixes
  • Particl Desktop
    • Buy flow
      • Extended the timeout duration during checkout
    • UI
      • Added hide-able page titles
      • Pre-release version determination updated to display warning
    • Listings
      • Sellers currently have the option of posting listings for duration varying between 1 day and 1 week
      • Added support for free shipping (shipping with a value of 0)
    • Overview
      • The spendable balance shown in the top bar of the application, when viewing a market related page, now displays the spendable Anon balance (was previously the spendable Blind balance)
      • The "Spendable" and "Locked" balances displayed on the Overview page now includes a summary of all balances
    • History
      • The transaction table replaces the "Balance Transfer" label with an indication of what the balance conversion was for (eg: "Converted -> Blind")
    • Orders
      • Added text to the help sections on the 'Orders' tab of the buy and sell pages to indicate to users that their wallets need to be unlocked in order to process buy flow messages (only applies if the wallet is encrypted)
    • Proposals
      • Proposals will now be visible for 7 days
      • A new error modal has been added to the Proposal creation flow indicating that an error has occurred, similar to the listing item creation error modal (replaces a toast style error message popup)
    • Bug fixes
  • Particl Copay
    • Cold Staking on Application
  • Particl Core
    • Release of 0.18.0.12 (release notes)
    • Release of 0.18.1.0 (release notes)
    • Release of 0.18.1.1 (release notes)
    • Release of 0.18.1.2 (release notes)
  • Third Party Integration
    • Trezor integration
